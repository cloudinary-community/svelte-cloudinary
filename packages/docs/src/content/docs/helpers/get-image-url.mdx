---
title: getCldImageUrl
---

You can use the `getCldImageUrl` helper to get a Cloudinary image URL outside the component. It uses the exact same API under the hood.

## Basic Usage

The required options are `width`, `height`, and `src`:

```ts
import { getCldImageUrl } from 'svelte-cloudinary';

const url = getCldImageUrl({
    src: 'images/turtle',
    height: 600,
    width: 960,
})
```
