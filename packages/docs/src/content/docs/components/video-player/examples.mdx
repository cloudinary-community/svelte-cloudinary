import { CldVideoPlayer, getCldImageUrl } from 'svelte-cloudinary';

## Basic Usage

Standard Cloudinary Video Player with playback.

<CldVideoPlayer
	config={{
		cloud: { cloudName: import.meta.env.VITE_CLOUDINARY_CLOUD_NAME },
	}}
	client:load
	width="1620"
	height="1080"
	src="videos/mountain-stars"
/>

```svelte
<script>
	import { CldVideoPlayer } from 'svelte-cloudinary';
</script>

<CldVideoPlayer
	width="1620"
	height="1080"
	src="<Your Public ID>"
/>
```

## Playback

### Adaptive Bitrate Streaming (ABR)

<CldVideoPlayer
	config={{
		cloud: { cloudName: import.meta.env.VITE_CLOUDINARY_CLOUD_NAME },
	}}
	client:load
	id="adaptive-bitrate-streaming"
	src="videos/mountain-stars"
	width="1620"
	height="1080"
	transformation={{
		streaming_profile: 'hd',
	}}
	sourceTypes={['hls']}
/>

```svelte {10-13}
<script>
	import { CldVideoPlayer } from 'svelte-cloudinary';
</script>

<CldVideoPlayer
	id="adaptive-bitrate-streaming"
	src="<Your Public ID>"
	width="1620"
	height="1080"
	transformation={{
		streaming_profile: 'hd',
	}}
	sourceTypes={['hls']}
/>
```

## Player Customization

### Chapters & Selector

Adding a button to select chapters.

<CldVideoPlayer
	config={{
		cloud: { cloudName: import.meta.env.VITE_CLOUDINARY_CLOUD_NAME },
	}}
	client:load
	id="chapters-selector"
	src="videos/mountain-stars"
	width="4096"
	height="2160"
	chapters={{
		0: 'Chapter 1',
		6: 'Chapter 2',
		9: 'Chapter 3',
	}}
	chaptersButton
/>

```svelte {10-15}
<script>
```

### Poster URL

Allows you to change the poster used for the player.

<CldVideoPlayer
	config={{
		cloud: { cloudName: import.meta.env.VITE_CLOUDINARY_CLOUD_NAME },
	}}
	client:load
	id="poster-custom-url"
	src="videos/mountain-stars"
	width="4096"
	height="2160"
	poster={getCldImageUrl({
		src: 'images/turtle',
	})}
/>

```svelte {10-13}
<script>
	import { CldVideoPlayer, getCldImageUrl } from 'svelte-cloudinary';

	id="poster-custom-url"
	src="<Your Public ID>"
	width="4096"
	height="2160"

	})}
```
```svelte {10-15}
/>
```

### Image Overlays

Place an image over the video (e.g. watermarks).

<CldVideoPlayer
	config={{
		cloud: { cloudName: import.meta.env.VITE_CLOUDINARY_CLOUD_NAME },
	}}
	client:load
	id="image-overlays"
	src="videos/mountain-stars"
	width="1620"
	height="1080"
	transformation={{
		overlay: 'images/earth',
		width: 150,
		gravity: 'north_east',
		x: 50,
		y: 50,
		opacity: 80,
	}}

## Advanced

### Localization

Provides localization options for player labels. Uses [Video.js JSON format](https://videojs.com/guides/languages/#json-format) to provide labels.

	config={{
		cloud: { cloudName: import.meta.env.VITE_CLOUDINARY_CLOUD_NAME },
	}}
	client:load
	id="localization"
	src="videos/mountain-stars"
	width="4096"
	height="2160"
	language="es"
	languages={{
		es: {
			Play: 'Reproducción',
			'Play Video': 'Reproduce el Video',
			Pause: 'Pausa',

			'subtitles off': 'Subtítulos desactivados',
